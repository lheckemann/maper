Generate an atlas for MAPER:

Create a list of atlas images, one image per line, full path. Save it to a file (e.g. images.txt)
Create a list of corresponding brain mask images (foreground 1, background 0). Save it to a file (e.g. masks.txt)
Create a list of corresponding segmentation files. Save it to a file (e.g. segs.txt)
Create a directory to hold the atlas, e.g. myatlas
Run
prep-db.sh -i images.txt -m masks.txt -s segs.txt -d myatlas


Prepare a set of target files for MAPER:

Create a list of target images, consisting of the full path.  Save it to a file (e.g. targets.txt)
Create a list of corresponding brain masks (foreground 1, background 0).  Save it to a file (e.g. masks.txt)
Create a directory to hold the target data set, e.g. mytargets
Run
prep-db.sh -i images.txt -m masks.txt -d mytargets


Run MAPER on a set of targets:

maper myatlas mytargets
